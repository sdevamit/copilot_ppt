Prompt Files

// Reusable, versioned slash commands

Location: .github/prompts/*.prompt.md

Markdown files with YAML frontmatter stored in your repo — versioned, reviewable, shareable. VS Code discovers them automatically as slash commands.

Agent, Variables & File References

Use agent: my-agent to lock the prompt to a custom agent. Add ${'{'}input:desc{'}'} for dynamic values and #file:path to inject live context.

Invoke from Chat or Programmatically

Type /prompt-name in Chat to run it, or use vscode.commands.executeCommand('workbench.action.chat.open', {prompt: '/name'}) from an extension.

.github/prompts/new-api-route.prompt.md
--- agent: db-expert tools: [codebase, editFiles, runCommand] description: Scaffold a new REST API route --- ## New REST Route: ${'{'}input:Route name, e.g. /orders{'}'} Create a typed Express route for `${'{'}input:Route name{'}'}`. Requirements: - Use Zod for request body validation - Add JSDoc with @param & @returns - Write 3 Vitest tests (happy path, validation fail, 500) - Follow standards in #file:.github/copilot-instructions.md
Chat: /new-api-route → prompts for route name → runs with DB Expert agent → generates route + tests
