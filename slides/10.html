Structural Decomposition

Best for ambiguous tasks. Ask Copilot to break down large features into sequential, implementable steps with clear owners.

What to Ask For

Request artifacts: migration checklists, dependency graphs, test strategies, rollback plans, and API schema changes.

Example Prompt

"Plan: Add cursor-based pagination to GET /users. Constraints: Postgres, <200ms p95. Produce steps, schema changes & rollout plan."

Strategic Outcome

Turns "unknowns" into a concrete roadmap. Use the output to generate PR descriptions or ticket requirements.
