Copilot Chat

Chat
Participants

Use @-mentions to route your prompt to the right context engine. These are chat participants — distinct from the Agent Mode sub-mode. Each expands what Copilot can see and act on.

Pro Tip

Combine participants + slash commands:
@workspace /explain @github list open bugs Works with Ask, Plan, and Agent modes

@workspace

Repository Context

REPO-AWARE

Answers questions using knowledge of the entire folder structure, file relationships, and symbols.

@workspace list call sites of fetchUser & find deprecated patterns.
@vscode

Editor Automation

IDE CONTROL

Controls VS Code UI. Can open files, create new files, run tasks, or change settings.

@vscode create src/services/user/mapper.ts with a typed skeleton.
@terminal

Shell Commands

CLI

Proposes shell commands to run in the integrated terminal. Great for git, npm, or migrations.

@terminal show command to run db migrations locally.
@github

GitHub Context

NEW

Query GitHub issues, pull requests, discussions, and repository data directly from your chat.

@github list open bugs assigned to me in this repo.
