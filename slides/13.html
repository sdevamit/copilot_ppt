13 / 20 Quick Reference: Agents & Slash Commands Your cheat sheet for Copilot Chat

Built-in Agents (@ prefix)

@workspace Searches your workspace using retrieval to surface relevant files, symbols, and structure. Context is retrieved — not the entire repo loaded at once.
@vscode Answers questions about VS Code settings, commands, extensions, and keyboard shortcuts.
@terminal Generates shell commands based on your current terminal context and OS.
@github Optional — availability depends on enabled integrations. Provides GitHub-aware Q&A (issues, PRs, repos) in supported setups.

Chat Variables (# prefix)

#fileAttach a specific file to your question for targeted analysis.
#selectionAutomatically includes the current editor selection as context.
#codebaseTells Copilot to search the full codebase for relevant context.
#terminalSelectionPasses the currently selected terminal output as context.

Slash Commands (/ prefix)

/explainWalk through selected code step-by-step — great for unfamiliar legacy code.
/fixIdentify and resolve bugs, logic errors, or compile issues in selected code.
/testsGenerate unit tests for the selected function or class, matching your testing framework.
/docAdd JSDoc, docstrings, or inline comments to undocumented code.
/optimizeSuggest performance improvements to selected code. (may vary by version)
/newScaffold a new project, file, or component. (may vary by version)
/newNotebookCreate a Jupyter notebook from a prompt. (may vary by version)
/setupTestsConfigure a test framework (Jest, Pytest, Vitest…). (may vary by version)
/searchNatural-language workspace search via @workspace. (may vary by version)
/clearStart a fresh chat session, removing prior conversation context.

Pro tip: Combine agents, variables, and slash commands for maximum precision. Example: @workspace /fix #file:auth.ts — Copilot searches the whole codebase to understand context, then applies a targeted fix to auth.ts only.  |  Use Tab to autocomplete agents and slash commands in the chat input box.
