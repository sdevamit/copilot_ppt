Context Health

Context Rot

When a long or noisy session causes Copilot's responses to drift, repeat, or contradict.

Context rot sets in when your chat window fills with stale history, failed attempts, or mixed topics — degrading the model's attention and output quality.

DO: Start fresh with /clear per task

One task, one session. Before switching topics, run /clear to reset the context window entirely.

DO: Use #file & #selection references

Reference files and selections instead of pasting code into chat. Less token noise, more precise context targeting.

DO: Prefer scoped context over @workspace

For targeted edits, use #file:path instead of broad workspace retrieval to keep the signal-to-noise ratio high.

DON'Ts

Mix unrelated topics in one session

Cross-topic sessions split the model's attention and produce hallucinations that blend both subjects.

Keep re-prompting a stuck session

After 2–3 failed attempts at the same goal, accumulated error turns make things worse. /clear and start over.

Paste the same code block repeatedly

Duplicate tokens consume the context budget without adding information. Use #file once instead.
